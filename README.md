# 万能表单

*By [酷飞软件](https://kufeisoft.com)*

## Installation

Use [Composer](https://getcomposer.org/) to install the library.

``` bash
$ composer require kufeisoft/form
```
## Basic usage

* 表单生成
``` php
\kufei\form::input();
```
* 表单验证
``` php
\kufei\form::validate();
```
